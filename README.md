## ✨ Key Features
- **Cosmic UI/UX:** Desain *glassmorphism* modern dengan perpaduan warna neon (Cyan & Purple).
- **Responsive Web Design:** Tampilan *pixel-perfect* di berbagai perangkat (Desktop, Tablet, dan Mobile).
- **Dynamic Typing Animation:** Efek teks mengetik yang dioptimalkan agar tidak menyebabkan *Layout Shift*.
- **SEO Optimized:** Dilengkapi dengan struktur semantik HTML5, Meta Tags, Open Graph, dan JSON-LD Schema Markup.
- **Integrated Support:** Terhubung langsung dengan *overlay pop-up* Trakteer untuk menerima apresiasi dari audiens tanpa meninggalkan halaman.

## 🛠️ Tech Stack
Proyek ini dibangun menggunakan fondasi web murni untuk memaksimalkan kecepatan dan pemahaman fundamental:
- **HTML5** (Semantic & Accessible)
- **CSS3** (Flexbox, CSS Grid, Custom Animations)
- **Vanilla JavaScript** (Intersection Observer API, DOM Manipulation)
